Objetivo del Proyecto

WireCalc GT es una herramienta educativa desarrollada para apoyar el aprendizaje de conceptos básicos relacionados con instalaciones eléctricas. El sistema permite estimar la corriente de una carga eléctrica y recomendar un calibre de conductor adecuado tomando en cuenta la potencia, el voltaje y la distancia del recorrido.

Este proyecto fue desarrollado como parte del curso de Introducción a Sistemas con el propósito de aplicar conocimientos de diseño web, navegación entre páginas y programación básica en JavaScript.
